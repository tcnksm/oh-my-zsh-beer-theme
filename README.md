Enjoy :beer: even while coding !

## Install

After installing [oh-my-zsh](), you can install this by:

```bash
$ curl -fsSL https://raw.github.com/tcnksm/oh-my-zsh-beer-theme/master/beer.zsh-theme >> ~/.oh-my-zsh/themes/beer.zsh-theme
```
